# Features
## Implemented
* Mizar commands
    * Mizar Compile
    * Run Mizar
    * Irrelevant Theorems
    * Irrelevant Iterative Steps
    * Irrelevant Inferences
    * Irrelevant Premises
    * Inaccessible Items
    * Trivial Proofs
    * Irrelevant Vocabularie
* syntax highlight
* auto indent
* hover information



## Under development
* go to definition
* auto completion

# Installation
1. Open VSCode and type "Ctrl+Shift+X".  
2. Type "mizar" in the search box and click to install.

# Usage
* Command Execution
    * Type "Ctrl+Shift+P" (or click title bar icon) and choose a command.
